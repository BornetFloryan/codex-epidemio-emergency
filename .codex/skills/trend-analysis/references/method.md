# Trend Method

This skill compares the two last numeric values of a time series.

## Rules

- If the last value is greater than the previous value, the trend is `increasing`.
- If the last value is lower than the previous value, the trend is `decreasing`.
- If both values are equal or almost equal, the trend is `stable`.

## Limits

- This is a simple comparison, not a statistical model.
- A reliable epidemiological trend needs several points, a clear period and source metadata.
- The result must not be presented as an official conclusion.
