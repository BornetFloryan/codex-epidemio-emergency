# Interprétation

Les indicateurs avancés sanitaires servent à repérer des signaux de surveillance.

Ils peuvent aider à :

- repérer une hausse potentielle ;
- identifier une source officielle ;
- préparer une note de situation ;
- orienter une veille sanitaire.

Un indicateur doit toujours être croisé avec :

- la période ;
- la zone géographique ;
- la méthode de collecte ;
- les autres sources officielles.
