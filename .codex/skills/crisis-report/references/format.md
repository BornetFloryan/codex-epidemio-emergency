# Situation Note Format

The crisis report skill returns a short operational note in JSON.

## Interpretation

- `attention_level` is a simple textual qualification based on the provided input.
- `signals_to_monitor` lists follow-up points for surveillance.
- `recommended_actions` gives cautious next steps, mostly source verification and context clarification.

## Limits

- The note is not an official public health assessment.
- The note does not diagnose individual medical situations.
- The user should verify official sources such as Sante publique France, Sentinelles and data.gouv.fr.
